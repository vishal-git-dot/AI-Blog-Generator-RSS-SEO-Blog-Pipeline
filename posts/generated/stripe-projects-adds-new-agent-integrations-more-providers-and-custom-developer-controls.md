# Stripe Projects adds new agent integrations, more providers, and custom developer controls

## Overview

Stripe is expanding its Projects platform with new agent integrations, 16 additional service providers (totaling 49), and developer controls for safer autonomous infrastructure provisioning. AI coding agents from Hermes, Factory Droids, and Warp can now directly provision infrastructure via Stripe Projects. New developer controls include unified spend visibility across providers, per-provider spending caps, isolated named environments (dev/staging/prod), and delegated authority for platforms to provision services on behalf of their developers. The goal is to bridge the gap where agents can write code but still struggle with adjacent tasks like credential management, account setup, and service connections.
