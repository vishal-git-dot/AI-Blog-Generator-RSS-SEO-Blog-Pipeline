# So I kept hitting dead proxy lists for Telegram -- all those

## Overview

So I kept hitting dead proxy lists for Telegram -- all those manual posts from years ago that don't work anymore. Got tired of it, so I wrote a scraper that pulls MTProto proxies from public Telegram channels, checks if they're actually alive with fake-TLS support, and keeps only the working ones. It runs automatically on a schedule, publishes fresh proxies to a Telegram channel, and maintains a simple web page with the current live list. No manual hunting, no stale links. If you need something similar or just want the proxies, it's all here: https://github.com/Yagami200/free-mtproto-proxies
