# Honestly, keeping MTProto proxies alive is a pain. Every lis

## Overview

Honestly, keeping MTProto proxies alive is a pain. Every list I found was either dead or required me to manually test each one. So I just automated the whole thing. The scraper sits in my cron, pulls proxies from public Telegram channels, tests each one for fake-TLS support, and drops anything dead before publishing. Only working entries make it into the final list. I also threw up a live web listing so you can grab the current set without running anything yourself. All the data is at https://github.com/Yagami200/free-mtproto-proxies . Still handy to curl the JSON directly if you're scripting something.
