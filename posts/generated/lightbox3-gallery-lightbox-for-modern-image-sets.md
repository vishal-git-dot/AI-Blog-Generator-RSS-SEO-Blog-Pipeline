# Lightbox3: Gallery Lightbox for Modern Image Sets

## Overview

Lightbox3: a JavaScript library that creates modern image and gallery lightboxes with no jQuery dependency. Features: Full-size image overlays from thumbnail links. Gallery grouping through HTML data attributes. Pinch zoom, swipe navigation, and swipe-to-dismiss. Spring-based open, close, and zoom animations.
