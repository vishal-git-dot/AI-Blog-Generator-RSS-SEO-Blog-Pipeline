# Building Fixit..

## Overview

and the map appears 👀 after a 1.5 month break because of semester exams, i'm finally back. i touched react again and man... i felt rusty as hell. for a moment i genuinely thought i forgot everything 😭. i couldn't even set up a fresh vite project, got frustrated, closed the laptop, went outside, came back, opened the docs instead of another tutorial and somehow got everything working myself lol. honestly, that felt way better than going back to tutorial hell. and now, fixit begins. fixit is a community based area issues reporting app. it's actually my second project idea. the first one is still staying a secret for now 👀. my first goal was getting the map up and running, and seeing it finally render felt pretty satisfying. also, today i learned that leaflet and react leaflet are actually two different things 😭. i seriously thought they were the same until i started reading the docs. from here, i'm planning to build this in public and be a lot more active on dev.to. i've got a lot more to build, a lot more to learn, and a lot more to post here as the project grows. i'm hoping this project pushes me to stay consistent, both with coding and writing. hopefully the next post isn't after another 1.5 month break 😂. see you in the next update :)
